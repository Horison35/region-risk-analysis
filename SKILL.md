# Анализ рисков регионов: Матрица Риск × Рейтинг РУСАДА

## Контекст и аудитория

Аналитический инструмент для приоритизации антидопинговой работы в субъектах РФ. Целевая аудитория — работники антидопинговой сферы. Объект анализа — **регионы (субъекты РФ)**, не виды спорта.

## Источник данных

| Источник | Содержание | Как получить |
|---|---|---|
| xlsx «Рейтинг регионов Итоги 2025» | ~89 регионов, 3 блока критериев, баллы 0–190 | https://rusada.ru/education/rating/ → «Рейтинг регионов Итоги 2025» |

```
https://rusada.ru/upload/iblock/5d3/7flsqyd9wfrrovtsuyc44yb9r04otp8u/Rei_ting-regionov-Itogi-2025.xlsx
```
Если URL изменился — найти актуальную ссылку через DOM-парсинг страницы rusada.ru/education/rating/.

## Структура рейтинга (3 блока, 190 баллов максимум)

| Блок | Критерии | Макс. |
|---|---|---|
| 1. Взаимодействие с РУСАДА | Ответственные/Сайт · Повышение квалификации · Соглашение · План-график · Отчёт · Опрос · Участие · Коммуникация · Мониторинг (штраф) · Организация* | 120 |
| 2. Образование в регионе | Квалификация лекторов · Обновление программ СШ · НМС для СШ · Соглашение о персданных · Уровень АД образования · Стенды | 100 |
| 3. Профилактика здравоохранения | Ответственный Минздрава · Программа для медперсонала · Межведомственное взаимодействие · Инновационные системы* | 50+30* |

**Важно:** xlsx содержит дублирующуюся строку-заголовок на строке 50 — фильтровать строки, где `Субъект РФ == "Субъект РФ"` или `ФО == "ФО"`.

## Логика зон риска

Прогнозная ipynb-модель для регионов отсутствует — зона определяется по итоговому баллу:

```python
# 🔴 КРИТИЧЕСКАЯ   — итого < 95  (<50% от 190)
# 🟠 УМЕРЕННАЯ     — 95 ≤ итого < 130
# 🟢 НИЗКАЯ        — итого ≥ 130
THRESHOLD = 130   # порог «высокий рейтинг»
MAX_SCORE = 190
```

## Матрица квадрантов (порог = 130 баллов)

| | Риск высокий (🔴/🟠) | Риск низкий (🟢) |
|---|---|---|
| **Рейтинг ≥ 130** | Усилить образование; беседы с ответственными; внесоревновательный контроль | Поддерживать; обмен лучшими практиками |
| **Рейтинг < 130** | Срочное вмешательство РУСАДА; приоритизировать невыполненные критерии | Профилактика; подтянуть рейтинг |

## Алгоритм выполнения

### 1. Скачать xlsx
```bash
wget -O /tmp/Reiting-regionov-Itogi-2025.xlsx \
  "https://rusada.ru/upload/iblock/5d3/7flsqyd9wfrrovtsuyc44yb9r04otp8u/Rei_ting-regionov-Itogi-2025.xlsx"
```

### 2. Запустить скрипт
```bash
python /home/ubuntu/skills/region-risk-analysis/scripts/build_report.py
```
Перед запуском обновить в начале скрипта: `XLSX_PATH`, `OUT_DIR`.

Скрипт автоматически:
- читает xlsx, пропускает строки-заголовки
- рассчитывает зоны риска и квадранты по всем регионам
- формирует Excel с листами «Матрица регионов» и «Сводка по ФО»
- строит Plotly HTML-дашборд (4 графика)

### 3. Самопроверка перед отправкой
- [ ] В итоговой таблице **все ~89 регионов** (нет строки «Субъект РФ»)
- [ ] Нет дублей — строка-заголовок на строке 50 xlsx отфильтрована
- [ ] Квадранты логически согласованы с баллами и зоной
- [ ] Дашборд: 4 графика, scatter с jitter, цвета по зонам
