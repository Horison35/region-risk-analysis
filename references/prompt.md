## Справочная логика (для понимания и проверки)

### Структура рейтинга (3 блока, 190 баллов максимум)

| Блок | Критерии | Макс. |
|---|---|---|
| 1. Взаимодействие с РУСАДА | Ответственные/Сайт · Повышение квалификации · Соглашение · План-график · Отчёт · Опрос · Участие · Коммуникация · Мониторинг (штраф) · Организация* | 120 |
| 2. Образование в регионе | Квалификация лекторов · Обновление программ СШ · НМС для СШ · Соглашение о персданных · Уровень АД образования · Стенды | 100 |
| 3. Профилактика здравоохранения | Ответственный Минздрава · Программа для медперсонала · Межведомственное взаимодействие · Инновационные системы* | 50+30* |

### Зоны риска

- 🔴 КРИТИЧЕСКАЯ — итого < 95 баллов (менее 50% от максимума)
- 🟠 УМЕРЕННАЯ — 95 ≤ итого < 130 баллов
- 🟢 НИЗКАЯ — итого ≥ 130 баллов

### Матрица квадрантов (порог = 130 баллов)

| | Риск высокий (🔴/🟠) | Риск низкий (🟢) |
|---|---|---|
| **Рейтинг ≥ 130** | Усилить образование; беседы с ответственными; внесоревновательный контроль | Поддерживать; обмен лучшими практиками |
| **Рейтинг < 130** | Срочное вмешательство РУСАДА; приоритизировать невыполненные критерии | Профилактика; подтянуть рейтинг |

### Обоснование к каждому региону

Формат: «Критический/умеренный риск из-за низкого итогового балла ([X] из 190); невыполненные критерии: [перечислить]». Для низкого риска: «Высокий уровень антидопинговой работы; выполнено [X] из ключевых критериев».

## Архитектура ответа

**Excel-файл `region_risk_final.xlsx`** с двумя листами:

Лист «Матрица регионов» — колонки:
- №
- ФО (Федеральный округ)
- Регион (Субъект РФ)
- Итого баллов
- % выполнения
- Место в рейтинге
- Зона риска

- Невыполненные критерии
- Рекомендация

**HTML-файл `region_risk_dashboard.html`** — интерактивный Plotly-дашборд.

Четыре графика:
1. Матрица рисков (scatter) — все регионы; jitter ±0.25 по Y; вертикальная линия порога 130; marker size ≈ 10, opacity ≈ 0.82
2. Топ рисковых регионов (горизонтальный бар, только 🔴/🟠) — marker_cornerradius=5, textposition="outside", cliponaxis=False
3. Распределение по квадрантам (pie)
4. Средний балл по федеральным округам (горизонтальный бар)
